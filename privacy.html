Política de Privacidade

Este aplicativo é utilizado apenas para integração com APIs do Google via OAuth 2.0, sem coleta ou armazenamento de dados pessoais.

Os dados acessados via autenticação Google são utilizados exclusivamente durante a sessão ativa do usuário no sistema.

Não compartilhamos dados com terceiros. Nenhuma informação é armazenada permanentemente.

Para mais informações sobre a API do Google, consulte: Política de Privacidade do Google
